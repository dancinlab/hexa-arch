## NUMB의 진짜 차별점 (honest)

```
✅ 진짜 우위 (이번 세션이 입증)         ⚠ 우위 아님 (정직)
─────────────────────────────         ─────────────────────
1. 소아 prilocaine-free 100× safe       · onset: Rx는 Pliaglis와 동급
   (EMLA MetHb wall 자체 제거)            (둘 다 lid 7+tet 7 — NUMB가
2. OTC+Rx 듀얼 SKU (단일 채널 탈피)        혁신한 게 아님)
3. 8 indication 제품군 매핑              · NOVEL 4종(N1·N6·N7)도
   (단품 → 범용)                          onset 아님 = depth/duration
4. epi headroom (LAST margin 273×)        (🔴 일관 반증)
5. 규제 최단 KR 일반의약품 ~$2M
```

**핵심**: NUMB는 "더 빠른 마취제"를 발명한 게 아니다 (onset은 Pliaglis 동급). 진짜 가치는 ① **소아 안전 재정의**(prilocaine-free) ② **OTC/Rx 동시 채널** ③ **범용 indication 매핑** ④ NOVEL 4종이 더한 **depth·duration** — 즉 "범용 + 안전 + 제품화" 축이지 onset 혁신이 아니라는 게 87-claim verify가 내린 정직한 결론.
