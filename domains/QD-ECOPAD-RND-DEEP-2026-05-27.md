### 1-2. 셀룰로오스 SAP first-principles (arxiv 1212.2426, Maurer 2012)

- 셀룰로오스 표면 분자 시뮬레이션 (DFT/MM 혼합)
- 수소결합 네트워크 강력 reconstruct (표면 첫 몇 층)
- **물 흡수 동역학**: 셀룰로오스 표면에서 물 분자 mobility ↓ (30°C 냉각 효과 · 10Å 깊이)
- **pure water만으로는 셀룰로오스 swelling 불충분** — meso-scale time
- → **흡수체 처방 최적화**: 셀룰로오스 + 친수성 보조 (예: 글리세린·xylitol·CMC)로 swelling 가속

### 1-3. 외음부 마이크로바이옴 안정성 모델 (arxiv 2206.10221, Greugny 2022)

| 발견 | 내용 |
|---|---|
| 모델 | ODE 기반 2-population (commensal + opportunistic) 마이크로바이옴 |
| 안정상태 1 | meta-stable (~2일) — 균형 |
| 안정상태 2 | 반전 stable (~300시간 = 12-13일) |
| **pH 상승 = dysbiosis 유도** | **표면 pH ↑** → opportunistic 우세 (C.acnes 또는 S.aureus) |

→ **ECOPAD 차별 가설**: 톱시트 pH 4.5-5.5 약산성 유지 + 항균 코팅 → 외음부 마이크로바이옴 안정성 유지 (락토바실러스 우세) · 12일 안정 모델 검증

### 1-4. 2017 릴리안 사태 후 한국 의약외품 강화

## 6. 🚀 진입 정공법 재정의 (Inertia 대비 한국·글로벌 동시)

```
Phase 1 (1-2개월·500-1,000만원)
  Pilot D 시제품 제작 (OEM) + 셀룰로오스 SAP swelling 자체 시험
  → 5초 흡수량 +30% vs Inertia 입증

Phase 2 (2-3개월·1,500-3,000만원)
  VOC 84 + 알러지 26 + 다이옥신 + 첩포 (KCL·KOTITI)
  → 의약외품 시험 패키지 완료

Phase 3 (4-6개월·800-1,500만원)
  외음부 마이크로바이옴 16S × 2 timepoint (n=44 cross-over)
  → 락토바실러스 우세 12일 입증 (Greugny 2022 모델 검증)

Phase 4 (2-3개월·인증 비용 500만원)
  인증 (GOTS·OEKO-TEX·EWG·비건협회) + 의약외품 신고
  → 한국 mid-price 슬롯 9-11k 진입 (해피문데이 14k vs 일반 7k)

Phase 5 (4-6개월·1,000-2,000만원)
  미국 FDA + 글로벌 진출 (Inertia·OCBON US와 직접 경쟁 또는 협력)

총: 13-20개월 · 4,300-8,000만원 (Phase 1-4 한국 진입)
```
