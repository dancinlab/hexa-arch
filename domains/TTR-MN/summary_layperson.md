# TTR-MN — 일반인 기준 요약 (시간 · 비용 · 통증 + 레이저 비교)

> milestone: TTR-MN.md @goal lay-person summary

## TL;DR

```
🩹 TTR-MN — "타투 지우개 패치"

- 하는 일: 손바닥만한 패치를 손가락으로 30초 누르면 미세 바늘이 녹으면서 약물 침투 → 잉크 분해
- 비유: 마치 비타민 패치 + 모기물린 후 가려움 패치 사이 — 30초 누르면 끝. 통증 없음.
        주 1회 × 52주 = 1년이면 90%+ 사라짐.
- 빈도: 주 1회 (5-10분/회) — 비타민 영양제 챙겨 먹듯
```

## 사용법 (한 사이클)

```
   [집에서 자가 적용]
   ① 패치 포장 뜯기                ┌────────────┐
   ② 타투 부위 깨끗이 닦기          │  TTR-MN    │
   ③ 패치 부착 + 손가락 30초 누름   │  ░░░░░░░░  │ ← 미세 바늘 (보일락말락)
   ④ 5-10분 대기 (바늘 녹음)        └────────────┘
   ⑤ 패치 제거 + 일상생활              ↓ 30초 press
                                  ⌜⌝⌜⌝⌜⌝⌜⌝   ← needles 진피 도달
   다음 cycle = 30일 후              ░░░░░░░░    ← HA + 약물 dissolution
                                  ⎽⎽⎽⎽⎽⎽⎽⎽    잉크 입자 분해
```

## TTR-MN vs 레이저 (일반인 비교)

| 비교 축 | 🔴 레이저 (PicoSure / Q-switched) | 🩹 TTR-MN 패치 (본 설계) |
|---|---|---|
| **1회 비용** | $300-500 (≈ 40-65만원) | **$20-40** (≈ 2.6-5만원) |
| **총 비용** | $2,400-6,000 (8-12회) | **$1,040** (1년 누적 52회) |
| **연 비용** | 첫 1-2년 집중 — $1,500-3,000/년 | **$1,040/년** (1년에 끝남) |
| **시술 횟수** | 8-12회 (6-8주 간격) | 52회 (주 1회 · 1년) |
| **1회 소요 시간** | 시술 5-15분 + 대기 30분 + 통원 1-2시간 | **5-10분** (집에서) |
| **장소** | 피부과 의원 · 의사 필요 | **자기 집** · OTC 또는 일반 처방 |
| **통증** | 강함 (고무줄로 튕기는 느낌 · 마취 크림 필요) | **거의 없음** (미세 바늘 · 짧고 얇음) |
| **다운타임** | 1-2주 (홍반 · 부풀기 · 딱지) | **없음** (즉시 일상 복귀) |
| **흉터 위험** | 있음 (잘못 시술 시 · 어두운 피부 위험 ↑) | 매우 낮음 (5-tier safety design 기준) |
| **색상 잉크** | 효능 약함 (흰·파랑·초록 어려움) | **F5 효소 · 색상 azo 잉크 강력** |
| **검정 잉크** | 효능 강함 (carbon 잉크 우선) | mid (DyP 효소 + macrophage 누적) |
| **시술 완료 시기** | 6개월-2년 (집중 시술) | **1년 cumulative** (주 1회 패치) |
| **부작용** | 색소침착 · 홍반 · 일시 흉터 · 화상 risk | aromatic amine 부산물 (M6 safety stacking 관리) |
| **임산부/수유부** | 금기 | TTR-MN 설계 시점 임상 데이터 미정 (Phase 1 후) |
| **숨겨진 비용** | 마취 크림 · 흉터 관리 · 색소침착 치료 | 패치 보관 (cool storage) · 가끔 의사 상담 |

## "52번 반복"의 의미 — 솔직 분석

```
   잉크 제거율 누적 (주 1회 기준 · per V3 multi-cycle + @goal v2)
   ─────────────────────────────────────────────────────────
   1주 (1회)      4.7%    ░░░░░░░░░░  (체감 거의 없음)
   4주 (4회)     17.6%    ▒▒░░░░░░░░  ("어 좀 흐려졌나?")
   13주 (13회)   47.0%    ▒▒▒▒▒░░░░░  (절반쯤 사라짐)
   26주 (26회)   71.8%    ▒▒▒▒▒▒▒░░░  (3분의 2)
   ★ 52주 (52회) 91.4%   ▒▒▒▒▒▒▒▒▒░  ★ 거의 사라짐 (1년)
```

- 비유: 마치 매주 한 번 비타민 패치 붙이듯 — 1년이면 90%+ 사라짐. 매니큐어는 1주, 타투는 1년.
- 비교: 레이저는 "**빨리 아프게 끝내기**" (1-2년 시술 + 다운타임) vs TTR-MN은 "**천천히 안 아프게**" (1년 매주 패치 · 통증/다운타임 없음)
