# 🧪 COSME R&D PLAYBOOK — 화장품 first-principles 효능 정공법

> 모든 `domains/COSME/` 자식 도메인은 본 PLAYBOOK 7-verb 정공법 + 화장품 바운더리 + 측정 방법을 따른다.
> 도메인별 차이는 **차별 가설·핵심 측정 지표·바운더리 매핑**에서만.

---

## 정공법 = 7-verb (demiurge 본연)

| 단계 | 의미 |
|---|---|
| 1. **spec** | 분류 바운더리 정밀 매핑 + 시판 TOP-N 벤치마크 |
| 2. **structure** | 활성성분 후보 + first-principles 효능 메커니즘 |
| 3. **design** | 처방 설계 (활성성분 × 전달체계 × 보조 시너지) |
| 4. **analyze ⟲** | pilot in vitro head-to-head + 차별 가설 정량화 |
| 5. **synthesize** | CGMP 시제품 제작 |
| 6. **verify** | 실측 검증 (in vitro·in vivo·인체적용시험 24주) + 통계 유의성 |
| 7. **handoff** | 화장품 신고 + (기능성 시) 심사 + 양산 + 채널 + 출시 |

```
시판 TOP (벤치마크) ──┐
                     │ → 동일 조건 시험 → 통계 유의성 (p < 0.05)
자체 R&D 시제품 ──────┘
                     ↓
                효능 +X% 우위 + 안전성 동등↑
                     ↓
              화장품 신고 (+ 기능성 심사) + 출시
```

## 화장품 분류 바운더리

| 항목 | 범위 |
|---|---|
| 법적 근거 | 화장품법 + 식약처 기능성 화장품 심사·신고 |
| 허용 클레임 (일반) | "청결·미용·체취 완화" 한정 |
| 허용 클레임 (기능성 11종) | 미백·주름개선·자외선차단·탈모완화·여드름완화·튼살완화·체취완화·염모·탈색·제모·물리적 자외선차단 |
| 시험성적서 | 안전성·전성분·VOC + 기능성 효능 시험 (in vitro/vivo) |
| 광고 금기 | "치료·예방" 의학적 효능 (의약외품/의약품 영역) |
| GMP | 화장품 GMP (CGMP) |

## 6대 고시성분 (탈모완화 자동 허가 루트)

```
조합①  덱스판테놀 0.2% + 살리실릭애씨드 0.25% + 엘-멘톨 0.3%
조합②  나이아신아마이드 0.3% + 덱스판테놀 0.5% + 비오틴 0.06% + 징크피리치온액(50%) 2%
```

- 정량 충족 시 **심사자료 면제** (사실상 신고제, 1-3개월, 수백만~수천만원)
- 6대 외 효능 주장 시 **인체적용시험 필수** (24주, 5천만~1.5억원)

## 공통 측정 방법

| 영역 | 측정 |
