# 💄 COSME — 화장품(cosmetics) 라인업 컨테이너

> 한국 약사법상 **화장품·기능성 화장품** 분류 라인업을 도메인 단위로 진행하는 컨테이너 폴더.
> QD(의약외품)과 분류 갈림에 따라 신설 — QD/는 의약외품, COSME/는 화장품 SSOT.

---

## QD vs COSME 분리 기준

| 축 | QD/ (의약외품) | COSME/ (화장품) |
|---|---|---|
| 법적 근거 | 약사법 · 식약처 의약외품 품목허가 | 화장품법 · 식약처 화장품 신고 |
| 효능 표시 | 예방·완화·각성 등 약효 표시 가능 | "청결·미용·체취 완화" 한정 |
| 진입 절차 | 품목허가 (3-12개월) | 제조판매업 신고 (1-2개월) |
| 광고 규제 | "치료" 표현 금지 (의약품 영역) | "의학적 효능" 표방 금지 (의약외품 영역) |
| 시험성적서 | 살균력·흡수량·pH 등 효능시험 | 안전성·전성분·VOC 등 |

## 진행 공통 단계 (4-funnel — QD/와 동일)

1. **discovery** — 시장·카테고리·경쟁 SKU·트렌드 스캔
2. **sourcing** — 제조사/OEM 발굴 + 화장품 제조판매업 신고 + (기능성 시) 기능성 화장품 심사
3. **listing** — 쿠팡 등록 + 광고 카피 점검 (의약품 효능 표현 차단)
4. **launch** — 발주·재고·정기배송·인플루언서 시드
