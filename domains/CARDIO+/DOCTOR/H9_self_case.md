## ◆ 4-axis 개인 적용 요약

> 한국인 PCI 환자(=본인)에게 4개 위험축이 동시에 작동할 수 있다. 각 축마다 (a) 근거 등급, (b) 지금 한국에서 가능한 것, (c) 주치의와 논의할 포인트.

| axis | 근거 등급 (verbatim) | 지금 한국에서 가능한 것 | 주치의와 논의할 포인트 |
|---|---|---|---|
| **① mPTP/IRI** (NOREFLOW) | mPTP IV bolus = 🔴 FALSIFIED (4 RCT) · 동아시아 nicorandil 6mo = 🔴 · NMN preconditioning = ⚪ | 시술 당시 adenosine IC + nicorandil 사용 여부 확인 가능 (한국 보험 routine) · 지금은 **표준 2차 예방**이 근거 중심 · NMN/NR 은 건강기능식품이나 심근보호 효과 **미확립** | 내 시술 기록상 no-reflow/IMR·adenosine/nicorandil 사용 여부 · NMN 복용이 근거 있는지 + 상호작용 |

## ◆ 개인 timeline (즉시 / 6m / 3y · 주치의 상의 전제)

```
 즉시   ┌─ ★ 3-axis 채혈 1회 패널 논의 (CYP2C19 PCR + Lp(a) + 선택 ALDH2*2)   ★ 최저 장벽
 (0m)   │   = "한 번 채혈"로 ②④(+①ethnicity) 동시 확보                       · 신규 IND/약 불요
        ├─ 현재 항혈소판제(주로 clopidogrel) 복용 지속 확인                   · 자의 중단 절대 금지
        └─ 표준 2차 예방 점검 (스타틴/LDL·혈압·금연·운동·당뇨)                · 모두 기존 급여
 6개월  ┌─ 검사 결과 기반 ② DAPT 적정성 재평가 (결정맵)                        ★ 중 (약제 재배치)
 (3-6m) ├─ ④ Lp(a) 결과 기반 LDL 목표 재설정 논의 (PCSK9 급여=LDL 기준)        · 변경은 주치의 판단
        └─ ③ ISR surveillance — 증상 점검 + 정기 외래 유지                    · 외래 NHIS 급여
 3년    ┌─ ④ Lp(a) 신약(siRNA/ASO) 한국 허가·급여 추적 (HORIZON 2026 H1)       ★ 최고 (신약 대기)
 (2-3y) ├─ ③ ISR 신규 옵션(fasudil-DCB 등) = 임상시험 단계                     · MFDS FIH
        └─ ① NMN-PRECON 등 IRB-light pilot = 설계 단계 (참고만)               · 효과 미확립
```

## ◆ CYP2C19 결과 해석 — NM/IM/PM 결정맵 (논의용)

> ⚠️ 주치의가 환자별로 판단하는 결정 트리의 일반 구조다. 본인에게 어떤 약을 쓸지는 출혈위험·시술복잡도·신기능·동반약을 종합한 **주치의 결정**이다. 자의로 약을 바꾸지 말 것.

```
  CYP2C19 PCR 결과
    ├─ NM/RM (~33%) ──► clopidogrel 대체로 적절 가능 · 환경변수 시 PFT 고려
    ├─ IM (~45%, 가장 흔함) ──► clopidogrel 부분 저하 가능 · 위험 낮으면 prasugrel 5mg
    └─ PM (~15%) ──► clopidogrel 크게 저하 가능 (ST↑) · prasugrel 5mg/ticagrelor
          · ★ ticagrelor 는 한국인 출혈위험↑ (TICAKOREA) → 출혈균형 핵심 (dual jeopardy)
          · PFT(VerifyNow PRU)로 잔존 효과 확인 후 조정
```

> 결정맵 적용 시 ST event 감소 X% = 🟠 (model assumption) / ⚪ — cohort outcome 미측정. 논의 포인트: "내 결과가 IM/PM 이면 현 clopidogrel 유지/변경/PFT 여부"를 출혈위험과 함께 주치의와 결정.

## ◆ d2 — 한국 미가용 옵션의 대안 path (포기 금지)

| axis | 한국 미가용 (wall) | 가용 대안 path (d2) |
|---|---|---|
| ① IRI | mPTP 직접약물 미수입 · elamipretide 미승인 | 표준 2차 예방 극대화 · NMN 은 상호작용만 주치의 확인 · ALDH2\*2 ethnicity-aware 맥락 |
| ② CYP2C19 | vicagrel 미도입 · selatogrel 연구단계 | **기존 급여 약제 내 최적화** (NM=clopi · IM/PM=prasugrel 5mg/ticagrelor) · PFT 보조 · vicagrel ~2028 추적 |
| ③ ISR | fasudil-DCB·dual-DCB = 연구단계 | **OCT/IVUS-guided 표준 재시술** (한국 세계 최고) · 표준 DCB · 위험인자 관리 |
| ④ Lp(a) | siRNA/ASO 미허가·비급여 (가장 큰 wall) | **PCSK9 mAb** (ASCVD 2차 급여, -25% 부수효과) · LDL 공격적 조절 · HORIZON 2026 후 신약/임상참여 추적 |

> 가장 큰 wall = ④ Lp(a) 신약. 그러나 d2 path 로 "PCSK9 + LDL 공격적 조절 + 향후 임상참여 추적"이라는 현 가용 대안이 분명히 존재 (포기 아님).
