## §0. 이 문서의 위치 + grounded/⚪ 경계 규약

본 X38 은 **X28 (index 환자 10년 Markov microsim)** 을 **30년 지평 + 차세대 도입 시점별 arm** 으로 확장한 **전망 시나리오 문서**다.
- X28 = "내 3축이면 **10년**에 얼마 차이?" → 실측(grounded base · +0.066 QALY/10y).
- **X38 = "그 10년 닻을 기준으로 **30년(65→95세)** 동안, 차세대 치료가 시점별로 도입되면 궤적이 어떻게 갈라질까?"** → ⚪ 전망.

> **운영화 = 강제 sim 아님.** 새 시뮬레이션·검증 실행 0건. X28 수치는 VERBATIM 인용(g5)하고, 30년/차세대는 전부 ⚪ 시나리오로 격리한다.

> - **확정**: A/B 의 0-10y 5개 값(생존·event-free·사망·QALY·LY)과 Δ(+0.066 QALY 등) — X28 실측.
> - **⚪ 전망**: "≈" 가 붙은 모든 칸. 10y 이후 외삽은 **선형 누적이 아니다** — 고령 사망률 자연 상승·competing risk 가 상대이득을 **희석**하므로, "10y Δ+0.066 × 3 = 30y Δ+0.2" 같은 단순 외삽은 **틀린 읽기**다.
> - arm C 추가분: muvalaplin/siRNA/VERVE 가 Lp(a) 를 더 깊게 낮춘다는 **방향**은 인과 β(LPA -0.3413 🟢, X28 grounding)와 정합하나, **MACE outcome 은 HORIZON(2026)·VERVE(2033+) readout 전까지 미확정** → arm C 우위의 "크기"는 ⚪ (d6).

### §2.1 왜 30년 누적이 "10y × 3"이 아닌가 (정직한 외삽 한계)

```
  단순 외삽(틀림)          현실적 ⚪ 전망(방향)
  ───────────────         ─────────────────────
  Δ10y = +0.066           65→75세: Δ ≈ +0.066 (X28 grounded)
  Δ20y = +0.132 (×2)  ✗   75→85세: 추가 Δ < +0.066 (고령 자연사망이 층화이득 희석)
  Δ30y = +0.198 (×3)  ✗   85→95세: 추가 Δ ≪ (competing-risk 지배 · 생존자 소수)
                          ⇒ 30년 누적 Δ는 +0.066 보다 크되 +0.198 보다 작을 가능성 ⚪
```

> **핵심**: 층화/차세대의 상대이득은 **이른 구간(아직 살아있고 위험축이 작동하는 시기)에 가장 크고**, 고령 구간엔 competing risk(다른 원인 사망)가 지배해 희석된다. 그래서 30년 누적은 **방향(B>A · C≥B)은 유지되나 크기는 단정 불가**다.

## §4. 차세대 도입이 "바꾸는 것" (정성 분석 · ⚪)
