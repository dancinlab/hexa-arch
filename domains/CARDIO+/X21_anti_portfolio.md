## §2. 포함 기준 (inclusion criteria) — 4-5 sub 의 공통 정의축 역도출

> 위 inventory 의 5열(scope·시간축·패널 axis·한국 leverage)에서 공통으로 추출되는 멤버십 조건. X17 §1.1 의 3 경계 기준을 X21 이 **포함기준(IC)** 으로 재명명·정밀화하고, 4개를 묶는 단일 응집축을 명시한다.

```
단일 응집축 (one organizing principle):
  "한 명의 한국 PCI 후 환자가 동시에 직면하는 잔여위험을,
   PCI 입원 중 채혈 1회 + 공유 카테터/imaging 인프라 한 지점에서 분기하여 다룬다."
  → CARDIO+ 는 임의의 cardio 주제 묶음이 아니라, 이 한 환자·한 채혈 시점에 수렴하는 축의 집합.
```

| IC | 포함 기준 (무엇을 묻는가) | 통과 (현 4-5 sub 의 충족 방식) | 위반 시 신호 |
|---|---|---|---|
| **IC-1** PCI-후속 특이성 | PCI(관상동맥 stent) 후 환자가 **직접 직면하는 잔여위험 축**인가 — 일반 심혈관 예방이 아니라 PCI-후속 특이 | N=급성 재관류손상 · D=혈전(항혈소판) · I=재협착 · L=지질잔여 · (AFPGX=삼중요법 출혈/혈전) | 일반 ASCVD 예방 / PCI 무관 / 환자군 분리 (예: 비-관상동맥 시술) |
| **IC-2** 한국/동아시아 leverage | ethnicity 빈도차 · NHIS 급여(기존 안에서 배포) · 글로벌 미시도(first-in-class) **중 ≥1 강함** | ALDH2\*2 40% · CYP2C19 40% · OCT 세계최고 · KIV-2 stratified primary 미시도 · (VKORC1 ~90%) | 한국 차별성 없음 / 글로벌 주도 / 이미 글로벌 포화 |
| **IC-3** 공유 인프라 결합성 | 채혈 1회 3-axis 패널 **또는** 공유 인프라(KAMIR registry · cath lab consortium · imaging · bio-verify-kernel)에 결합 가능 | 채혈 패널 1 SNP/측정 추가로 통합 · 동일 PCR lab · 공통 cath lab/imaging (X9 §5/§6 · X11) | 별도 인프라 필요 / 패널·registry·cath lab 무관 / 독립 자산 요구 |
| **IC-4** 비-cannibalization | 기존 4 sub 와 **중복되지 않는 고유 축**인가 (직교성) | N=손상 · D=항혈소판 · I=재협착 · L=지질 → 상호 직교 · (AFPGX=항응고, DAPT와 직교) | 기존 도메인이 이미 점유한 축 (예: Lp(a) 외 일반 지질 → LPA 중복) |

> **포함 판정 규칙 (기준-기반)**: 4 IC 를 **모두** 강하게 충족할수록 멤버십 강함. IC-1(PCI-후속)이 **필수 게이트**(위반 시 무조건 경계 밖) · IC-2/IC-3/IC-4 는 강도 합산. X17 §3 점수(② 4.8 · ③ 4.6 · ① 3.7 · ⑥ 3.1 · ⑤ 2.4 · ④ 2.3)는 이 IC 들의 ⚪ 주관 가중 합산이었다 — X21 은 그 **점수의 임계선**을 정의한다 (§4 경계 ASCII).

## §3. anti-portfolio 표 — 명시적 제외 영역 (왜 안 넣었나)

> "cardio 이지만 CARDIO+ 경계 밖"인 영역. 각 행 = 어느 IC 를 위반했는가(기준-기반) + 어디 속하나 + 경계선 여부. 제외 판단의 근거는 IC(§2) 위반이며, 정도(degree)·"별도 메타도메인 후보냐"의 가중은 ⚪ 주관(명시).

| 제외 영역 | 왜 제외 (위반 IC · 기준-기반) | 어디 속하나 (별도 메타도메인?) | 경계선 사례? |
|---|---|---|---|
| **TAVR / 구조심장 (판막)** ★대표 | **IC-1 위반 (필수 게이트)** — TAVR 는 대동맥판막 중재이지 관상동맥 PCI(stent)가 아니다. 환자군·시술·잔여위험 축(판막혈전·전도장애·PVL)이 PCI-후속과 다름. **IC-3 위반** — 채혈 3-axis 패널 무관(cath lab 물리공간 일부 공유뿐). | **별도 메타도메인** `STRUCT+`(구조심장) — 독립 도메인이 옳음 ⚪ | ❌ 아니오 — IC-1 명확 위반, 경계 밖 단호 |
| **HF (post-MI 심부전 · SGLT2i/ARNI)** | **IC-2 약** — SGLT2i/ARNI 는 글로벌 주도(DAPA-HF·PARADIGM·EMPEROR) · ethnicity 차 미약 · 미시도 leverage 약. **IC-3 약** — 채혈 패널 axis 아님(LVEF/NT-proBNP = 영상·혈액 surrogate). IC-1 은 부분 충족(post-MI 잔여)이라 완전 배제 아님. | **별도 HF 도메인**(예 `HF+`) — KAMIR/imaging consortium 결합은 가능하나 패널 분기점이 아님 ⚪ | ⚠ **약한 경계선** — IC-1 통과·IC-2/IC-3 약. 별도 도메인이 적합 신호(X17 ① 3.7) |
| **AF ablation (전기생리 · EP)** | **IC-1 위반** — ablation 은 부정맥 전기생리 시술이지 PCI-후속 잔여위험이 아니다(시술·장비·환자 path 분리). **IC-3 위반** — 채혈 PGx 패널·cath lab(중재) 인프라와 별개 EP lab. ※ AF 동반 환자의 **항응고 PGx(AFPGX)** 는 IC 충족(§1 5th)이나, ablation 시술 자체는 경계 밖 — **AF 약물(항응고) ⊂ 안 vs AF 시술(ablation) ⊂ 밖** 분리. | **별도 EP 메타도메인** — AF 관리 중 약물축만 AFPGX 로 흡수, 시술축은 분리 ⚪ | ⚠ **분할 경계선** — 같은 질환(AF)이 약물축(안)·시술축(밖)으로 갈림 |
| **판막 (일반 valvular · 협착/역류 surveillance)** | **IC-1 위반** — 판막질환은 관상동맥 PCI 와 별개 병태·시술(TAVR/SAVR/MitraClip). **IC-3 위반** — 패널 무관. (TAVR 의 상위 카테고리.) | **별도 구조심장 메타도메인** `STRUCT+` (TAVR 와 동일 home) ⚪ | ❌ 아니오 — IC-1 명확 위반 |
| **CAC / coronary imaging (석회화 score · CT)** | **IC-3 약** — 영상축이라 채혈 패널 결합 약함(CT/OCT). **IC-2 모호** — CAC 영상 인프라 있으나 ethnicity 차 신호 약·미시도 모호. IC-1 부분 충족(PCI 시 석회화=IVL/rotablation 직결)이나 "PCI 후 결정 변경" evidence 중간. | **ISR sub + imaging consortium 흡수** — 독립 sub-domain 보류, ISR 의 보조 영상 sub-track 으로 흡수가 적합 ⚪ | ⚠ **강한 경계선** — IC-1 통과·IC-3 약. ISR/imaging 자산으로 흡수냐 독립이냐가 진짜 애매(X17 ⑥ 3.1) |
| **일반 LDL/TG 지질 (PCI-비특이 ASCVD 2차예방)** | **IC-4 위반 (cannibalization)** — Lp(a) 잔여위험은 LPA 가 이미 점유. **IC-1 약** — LDL/TG 는 PCI-특이가 아닌 일반 ASCVD 2차예방. **IC-2 약** — 한국 차별성 낮음(글로벌 IMPROVE-IT/REDUCE-IT 성숙). ※ SLCO1B1(스타틴 근병증)은 PGx add-on 으로 IC 충족(§1) — **약물유전체 축 ⊂ 안 vs 일반 LDL 목표치 관리 ⊂ 밖** 분리. | **LPA sub 내 보조항목** 또는 **일반 예방(메타도메인 외)** — 신규 지질 sub-domain 은 LPA 와 중복 ⚪ | ❌ 대체로 아니오 — IC-4 명확. 단 SLCO1B1 PGx 만 경계 안쪽으로 분리됨 |

### 3.1 제외 판정 근거 1줄 메모 (기준-기반 · d6 — 억지 분류 강제 0건)

- **TAVR/판막/구조심장 (가장 명확한 제외)**: IC-1(PCI-후속) **필수 게이트** 위반 — 관상동맥 stent 가 아니다. 경계선 아님(단호). X17 ④ 2.3 · A축 최저(1점)와 정합.
- **HF (post-MI)**: IC-1 은 통과(post-MI 잔여위험)하나 IC-2(글로벌 주도)·IC-3(패널 비결합)이 약해 별도 HF 도메인이 적합 — 약한 경계선.
- **AF ablation**: 같은 질환(AF)이 **약물축(AFPGX, 안) vs 시술축(ablation, 밖)** 으로 갈리는 분할 경계선. 항응고 PGx 만 채혈 패널 4-axis 로 흡수, ablation 시술은 EP 메타도메인.
- **CAC/imaging**: IC-1 통과·IC-3 약 — **ISR/imaging consortium 으로 흡수냐 독립 sub 냐**가 가장 애매한 강한 경계선(독립 도메인 보류).
- **일반 LDL/TG**: IC-4 cannibalization(LPA 중복) — 단 SLCO1B1 PGx 축은 IC 충족으로 분리되어 경계 안쪽(패널 add-on).

> ⚪ **honest fence**: 위 "약한/강한/분할 경계선" 정도(degree) 판단과 "어느 별도 메타도메인에 속하나"의 가중은 X21 의 주관(⚪)이다. IC 위반 여부 자체는 §2 의 조직 원리(X17 §1 carry)에 근거한 기준-기반 분류다. literature 성숙도(guideline 정착 등)는 일반 evidence-level 묘사이며 sub-V verdict 부여 X.

## §5. 메타 함의 — `+` 메타도메인의 응집성(coherence) 원칙

> 무엇이 하나의 `+`(CARDIO+)를 묶는가. domain skill 의 `+`(meta-domain composition) 는 "관련 주제를 모으는 폴더"가 아니라 **단일 수렴점(convergence point)을 공유하는 축들의 집합**이어야 한다.

```
응집성 원칙 (coherence principle for `+`):
  ① 단일 수렴점이 있다           — CARDIO+: "한 한국 PCI 환자 · 채혈 1회 + 공유 cath lab/imaging"
                                    (sub 들이 한 점에서 분기 = leverage 의 원천)
  ② sub 들이 상호 직교한다        — N·D·I·L·(A) 가 서로 다른 잔여위험 축 (중복 0 · IC-4)
                                    → 직교해야 cannibalization 없이 누적 가치
  ③ 단일 입력 자산이 다축 동시 unlock — 채혈 1 PCR(§X9 6 wall) · 단일 hexa-lang PR(4 도메인 🔵)
                                    · KOVA WGS 1건(3 도메인 genotyping) → cross-domain leverage
  ④ 경계(밖)가 명시적으로 정의된다  — 무엇을 제외하는가(§3)가 곧 무엇이 묶는가의 정의 (anti-portfolio)
```

| 응집성 신호 | CARDIO+ 의 충족 | 만약 위반하면 (anti-pattern) |
|---|---|---|
| 단일 수렴점 | 채혈 1회 3-axis 패널 + 8-기관 cath lab/imaging consortium (X9 §5/§6) | 폴더에 cardio 주제만 모으면 leverage 없는 단순 집합 (예: TAVR 추가 시 수렴점 깨짐) |
| 직교성 (IC-4) | N=손상·D=항혈소판·I=재협착·L=지질·A=항응고 상호 직교 | 일반 LDL 추가 시 LPA 와 중복 → 누적가치 0 (cannibalization) |
| 다축 동시 leverage | 단일 PR/단일 채혈/단일 WGS 가 여러 wall 동시 해소 (X9 §7) | 각 sub 가 독립 인프라 요구 시 `+` 의 의미 소실(분리 도메인이 옳음) |
| 명시적 경계 (anti-portfolio) | TAVR/HF/ablation/판막/CAC/일반지질 제외 명문화 (§3) | "cardio 면 다 넣음" → scope creep · 응집성 희석 |

> **메타 함의 1줄**: `+` 메타도메인의 응집성은 sub 의 **주제 유사성이 아니라 단일 수렴점 공유 + 상호 직교성**에서 온다. CARDIO+ 가 응집적인 이유는 4-5 sub 가 모두 "한 한국 PCI 환자가 채혈 1회 + 공유 cath lab/imaging 한 점에서 분기하는 직교 잔여위험 축"이기 때문이며, TAVR/HF/ablation 을 제외하는 행위(anti-portfolio)가 바로 이 수렴점을 보존하는 경계 정의다. 무엇을 제외하는가가 무엇이 `+`를 묶는가를 정의한다.

## §7. X21 산출 요약

- **포함 기준 (inclusion criteria · 역도출)** = 4 IC — **IC-1** PCI-후속 특이성(필수 게이트) · **IC-2** 한국/동아시아 leverage(ethnicity·NHIS급여·미시도) · **IC-3** 공유 인프라 결합성(채혈 1회 패널 · cath lab/imaging consortium) · **IC-4** 비-cannibalization(직교성). 단일 응집축 = "한 한국 PCI 환자 · 채혈 1회 + 공유 인프라 한 점 분기".
- **anti-portfolio (명시적 제외)** = TAVR/판막/구조심장(IC-1 게이트 위반) · post-MI HF(IC-2/IC-3 약) · AF ablation(IC-1 위반, 시술축만) · CAC/imaging(IC-3 약) · 일반 LDL/TG(IC-4 cannibalization).
- **가장 명확한 제외** = **TAVR(구조심장 판막)** — IC-1(PCI-후속) 필수 게이트 명확 위반(관상동맥 stent 아님 · 채혈 패널 무관). 별도 `STRUCT+` 메타도메인이 옳음. 경계선 아님(단호).
- **경계선(애매) 사례** = **CAC/coronary imaging** — IC-1 은 통과(PCI 시 석회화=IVL/rotablation 직결)하나 IC-3(채혈 패널 비결합, 영상축) 약해 **ISR/imaging consortium 흡수냐 독립 sub 냐**가 진짜 애매(X17 ⑥ 3.1, 독립 도메인 보류). 분할 경계선으로는 **AF**(약물 항응고축=안 / ablation 시술축=밖).
- **메타 함의 (`+` 응집성 원칙)** = `+` 메타도메인은 주제 유사성이 아니라 **단일 수렴점 공유 + sub 간 직교성**으로 묶인다. 무엇을 제외하는가(anti-portfolio)가 무엇이 `+`를 묶는가를 정의한다 (수렴점 보존이 곧 경계).
- **g5/d6/d1 준수**: 기존 4 grade X8 VERBATIM(재판정 0) · 제외 판정 IC 위반 기준-기반 · degree/좌표/원칙 일반화는 ⚪ 주관 명시 · X17 점수 carry(재산출 0) · LLM self-judge 0 · 억지 분류 강제 0 · 새 verify/sim/atlas/pool/cloud/python 0건 · git 0건.

→ **X17 vs X21 분담**: X17 = "5th 후보 6개 점수화 → AFPGX 추천" (확장 방향) · X21 = "포함기준 역도출 + 제외 카탈로그 + `+` 응집성 원칙" (경계/제외 원칙, 다른 각도). 다음 단계 — 5th(AFPGX) 신설 시 §1 inventory 행 확정 + §4 경계 ASCII "안" 영역 갱신 · CAC 경계선은 ISR imaging sub-track 으로 흡수 결정 시 §3 행 이동.
