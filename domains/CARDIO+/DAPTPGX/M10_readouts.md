## §10. 핵심 통찰 (5 bullets)

- **2024-2026 readout의 한국 임팩트 중심축은 "chronic clopi 단독 우위 강화"** — HOST-EXAM 10y (HR 0.86, n=5,438) + SMART-CHOICE 3 (HR 0.71, n=5,506) 두 한국 RCT가 동시 출판되며 chronic phase column이 evidence-leading 영역 확정. M7 §6.4 timeline의 baseline 자체가 한국 우위로 재calibration — ESC/ACC 2027 update에서 한국 데이터 인용 가능성.

- **ACC/AHA 2025 (Rao)의 ticag monotherapy ≥1mo Class IIa 격상이 글로벌 단축-DAPT 패러다임 전환의 정점** — ULTIMATE-DAPT + TARGET-FIRST + T-PASS 누적 근거로 "1mo→P2Y12 mono"가 표준 옵션 진입. 단 한국 결정맵은 TICAKOREA 출혈 시그널 (HR 2.26) reconcile 필요 — KSC 2025는 ticag mono를 Class IIa로 받되 한국인 dose/duration 별도 layer 유지 권고 ⚪.

- **CELEBRATE (zalunfiban) 양성이 prehospital 카테고리 본격 개막** — adj OR 0.79 + STR 40.5/33.1% + bleeding 동등 → SC αIIbβ3 prehospital 첫 양성 phase 3 RCT. SOS-AMI (selatogrel) readout 2026과 합쳐지면 2027-2028에 STEMI 결정맵에 "prehospital SC bridge" 가지 신설 — M6 §5·M7 §6.2 가설이 임상 데이터로 전환되는 시점.

- **vicagrel FDA NDA 2023-12 filing이 한국 PGx 결정맵 최단기 게임 체인저** — FDA 결정 2026 + NMPA 진행 병행 → 글로벌 라이센스 deal 트리거. 한국 IM+PM 60% (~60명/100 PCI)에 1:1 clopi 대체 가능. M6 §4·M7 §6.1 가설의 "2028 KFDA" timeline이 1-2y 단축 가능성. 단 CSPC↔Jiangsu Vcare↔글로벌 파트너 dealing 변수 ⚪.

- **2024-2026 음성 readout 2건이 신약 pipeline 정리** — ACTISAVE (glenzocimab AIS, 2024) + BMS-986141 stroke arm 중단 (n=16). M6 §5·§6 인접 표적 (GPVI · PAR-4) 단기 임상 진입 가능성 ↓ → 결정맵 4D tesseract 확장 timeline은 2030+ 후퇴. **현실적으로 2026-2028 단기 결정맵 evolution은 selatogrel/vicagrel/zalunfiban 3제 진입 여부에 집중** — M6 multiplex 가설의 첫 검증 라운드.
