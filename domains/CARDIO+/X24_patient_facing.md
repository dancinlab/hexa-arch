## §1. 큰 그림 — "왜 유전자 3가지를 보나요?"

심장 혈관을 넓히는 시술(PCI)을 받은 뒤에도, 우리 몸은 **타고난 체질**에 따라 반응이 조금씩 다릅니다.
같은 약, 같은 관리를 해도 사람마다 결과가 다른 이유 중 하나가 바로 이 "타고난 체질"입니다.

이 자료는 그 체질을 3가지 검사로 나눠 봅니다. 어렵게 들리지만, 비유로 보면 쉽습니다.

```
   내 심장 시술 후 "타고난 체질" 3가지 (한 번 채혈로 다 확인 가능)
   ────────────────────────────────────────────────────────────

   🔌  CYP2C19   = "약을 켜는 스위치가 약한 체질"
                   → 피 굳음 막는 약이 잘 안 켜질 수 있어요

   🍺  ALDH2     = "술 마시면 빨개지는 유전자" (참고용)
                   → 같은 유전자가 심장 혈관에도 영향을 줄 수 있어요

   🍯  Lp(a)     = "타고난 끈끈한 콜레스테롤"
                   → 운동·식사·일반 약으로 잘 안 줄어드는 위험 한 조각

   ※ 셋 다 "한 번 피 뽑기"로 확인됩니다. 새 약 승인이나 입원이 필요 없어요.
```

> 한국 사람은 이 3가지를 **동시에** 가지고 있을 확률이 서양 사람보다 높습니다. 그래서 한 번에 같이 보는 게 효율적입니다.

## §2. 내 유전자 검사 3가지 — 비유로 쉽게

### 🔌 CYP2C19 — "약을 켜는 스위치가 약한 체질"

- **쉬운 이름**: 약 스위치 (원래 이름: CYP2C19, 씨와이피 투씨 나인틴)
- **하는 일**: 어떤 약은 몸에 들어와서 "켜져야" 효과가 납니다. 이 유전자가 그 **스위치** 역할을 합니다.
- **왜 중요한가**: 심장 시술 후 흔히 먹는 **클로피도그렐**(피 굳음을 막아 스텐트가 막히지 않게 하는 약)은
  몸 안에서 켜져야 듣습니다. 스위치가 약하면 약을 먹어도 **덜 켜져서** 효과가 떨어질 수 있어요.
- **비유**: 리모컨에 약한 건전지를 끼운 것과 같습니다. 버튼(약)을 눌러도 신호(효과)가 약하게 갑니다.

```
   약(클로피도그렐) 먹음
        │
        ▼
   ┌──────────────┐
   │  🔌 스위치    │   강한 스위치(NM) → 찰칵! 약이 잘 켜짐 ✅
   │  (CYP2C19)   │   약한 스위치(IM/PM) → 깜빡... 약이 덜 켜짐 ⚠️
   └──────────────┘
        │
        ▼
   피 굳음 막는 효과
```

> 한국인 약 60%가 스위치가 "보통보다 약한" 쪽(중간 또는 약함)에 속한다고 알려져 있습니다.
> (연구 인용 수치 — 약함=15%쯤, 중간+약함 합쳐 63%쯤. 이 글이 정한 게 아니라 연구에서 그대로 가져온 값입니다.)

### 🍺 ALDH2 — "술 마시면 빨개지는 유전자" (참고용)

- **쉬운 이름**: 술 빨개짐 유전자 (원래 이름: ALDH2, 알데하이드 분해효소 2)
- **하는 일**: 술을 분해하는 유전자입니다. 이게 약하면 술 한 잔에도 얼굴이 금방 빨개집니다.
- **왜 심장에서?**: 같은 유전자가 혈관을 넓히는 몸속 신호(산화질소)에도 관여합니다. 그래서 심장 시술 후
  혈관 반응에 **영향을 줄 수 있다**고 봅니다. 다만 이건 아직 **연구 단계**라, 약을 정하는 데 바로 쓰지는 않습니다.
- **비유**: 얼굴이 빨개지는 건 "이 사람은 분해 효소가 약하구나"를 알려주는 **빨간 신호등**입니다.
  그 신호등이 심장 혈관 쪽에서도 살짝 켜져 있을 수 있다는 뜻입니다.

```
   술 한 잔  →  😳 얼굴 빨개짐   = ALDH2 약함 신호
                     │
                     └─ (참고) 심장 혈관 반응에도 영향 가능 — 아직 연구 단계
```

> ⚠️ 이건 **참고 정보**입니다. "빨개지니까 심장약을 바꿔야 한다" 같은 결론으로 이어지지 **않습니다.**

### 🍯 Lp(a) — "타고난 끈끈한 콜레스테롤"

- **쉬운 이름**: 끈끈이 콜레스테롤 (원래 이름: Lp(a), 리포단백질 a)
- **하는 일**: 콜레스테롤의 한 종류인데, **태어날 때 양이 거의 정해집니다.** 평생 잘 안 변합니다.
- **왜 중요한가**: 운동·식사 조절이나 흔한 콜레스테롤 약(스타틴)으로도 **잘 안 줄어듭니다.** 그래서
  다른 관리를 잘 해도 **남아 있는 위험 한 조각**이 됩니다.
- **비유**: 일반 콜레스테롤이 청소 가능한 먼지라면, Lp(a)는 **타고난 끈끈한 껌**과 같습니다. 비질(스타틴)로는
  잘 안 떨어집니다. 대신 **딱 한 번만 재면** 평생 값을 알 수 있어요 (거의 안 변하니까).

```
   일반 콜레스테롤 (먼지)   →   비질(스타틴)로 잘 쓸림 ✅
   🍯 Lp(a) (끈끈한 껌)    →   비질로는 잘 안 떨어짐 ⚠️  (타고난 양 · 평생 거의 그대로)

   ※ 그래서: 평생 딱 1번만 재면 충분합니다.
```

> 한국인 약 12~13%가 "높은 편"(50 이상)이라고 알려져 있습니다 (연구 인용).

## §3. "내 결과가 이러면 무슨 뜻일까?" — 평이 표

> 아래는 **일반적인 안내**입니다. 실제로 무엇을 할지는 **출혈 위험, 시술 종류, 콩팥 기능, 같이 먹는 약**까지
> 다 따져서 **주치의가 정합니다.** "내 결과 → 의사가 고려할 것"의 큰 그림만 보세요.

| 검사 | 내 결과가 이러면 | 의사가 고려할 것 (의사 판단) |
|---|---|---|
| 🔌 **CYP2C19** (약 스위치) | 스위치 강함 (정상) | 지금 약(클로피도그렐)이 잘 맞을 가능성 — 대체로 유지 |
| | 스위치 중간 (한국인 가장 흔함) | 약이 **덜 들을 수 있음** → 위험 낮으면 다른 약을 검토하거나, 약효 검사를 볼 수 있음 |
| | 스위치 약함 | 약이 **많이 덜 들을 수 있음** → 다른 약 검토 · 단, 다른 약은 출혈이 더 날 수 있어 **균형**을 봄 |
| 🍯 **Lp(a)** (끈끈이) | 낮음 (50 미만) | 평소 관리(콜레스테롤 목표) 유지 — 다시 잴 필요 없음 |
| | 높은 편 (50 이상) | "남은 위험"이 크니 콜레스테롤을 **더 강하게** 낮추는 걸 검토 |
| | 아주 높음 (75 이상) | 가장 높은 남은 위험 → 앞으로 나올 신약·임상시험 정보를 의사와 함께 추적 |
| 🍺 **ALDH2** (술 빨개짐, 참고) | 약함 (빨개지는 체질) | 체질 참고 정보로 인지 — **약을 바로 바꾸는 근거는 아님** (연구 단계) |

> 💡 한 가지 중요한 점: 끈끈이(Lp(a))가 높아도, 그 자체로 바로 신약을 쓰는 건 아닙니다. 한국 건강보험에서
> 강력한 콜레스테롤 주사약은 **LDL("나쁜" 콜레스테롤) 수치 기준**으로 정해집니다. Lp(a)가 높으면 "더 신경 쓸 이유"가
> 되는 것이지, 그 숫자 하나로 약이 정해지지는 않습니다. 이 부분은 의사와 꼭 같이 보세요.

## §4. 셋을 한 그림으로 — "한 번 채혈, 세 가지 체질"

```
                          🩸  피 한 번 뽑기
                                 │
          ┌──────────────────────┼──────────────────────┐
          ▼                      ▼                      ▼
     🔌 CYP2C19             🍺 ALDH2 (참고)          🍯 Lp(a)
     "약 스위치"            "술 빨개짐"              "끈끈이 콜레스테롤"
          │                      │                      │
     약이 잘 켜질까?         혈관 반응 참고           평생 남는 위험 한 조각
          │                      │                      │
          ▼                      ▼                      ▼
     의사와 "내 약이          (체질 인지용 ·          의사와 "콜레스테롤
      잘 맞나" 상의           연구 단계)              목표를 더 낮출까" 상의

   ※ 이 그림은 "이렇게 하라"가 아니라 "무엇을 의사와 의논하면 되는지"의 지도입니다.
```

비유로 정리하면:

```
   🔌 스위치  : 리모컨 건전지가 강한가 약한가 (약이 잘 켜지나)
   🍯 끈끈이  : 비질로 안 떨어지는 타고난 껌 (평생 1번만 재면 됨)
   🍺 빨개짐  : 술자리의 빨간 신호등 (참고 정보)
```
