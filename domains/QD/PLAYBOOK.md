# 🧪 QD R&D PLAYBOOK — 의약외품 first-principles 효능 정공법

> 모든 `domains/QD/` 자식 도메인은 본 PLAYBOOK 7-verb 정공법 + 의약외품 바운더리 + 측정 방법을 따른다.
> 도메인별 차이는 **차별 가설·핵심 측정 지표·바운더리 매핑**에서만.

---

## 정공법 = 7-verb (demiurge 본연)

| 단계 | 의미 |
|---|---|
| 1. **spec** | 분류 바운더리 정밀 매핑 + 시판 TOP-N 벤치마크 (효능 지표·처방·가격) |
| 2. **structure** | 활성성분 후보 + first-principles 효능 메커니즘 분석 |
| 3. **design** | 처방 설계 (활성성분 농도 × 전달체계 × 보조성분 시너지) |
| 4. **analyze ⟲** | pilot in vitro head-to-head (시판 vs 자체) + 차별 가설 정량화 |
| 5. **synthesize** | GMP 시제품 제작 |
| 6. **verify** | 실측 검증 (in vitro·in vivo·임상) + 통계 유의성 |
| 7. **handoff** | 분류 신고/허가 + 양산 + 채널 (쿠팡/약국) + 출시 |

```
시판 TOP (벤치마크) ──┐
                     │ → 동일 조건 시험 → 통계 유의성 (p < 0.05)
자체 R&D 시제품 ──────┘
                     ↓
                효능 +X% 우위 + 안전성 동등↑
                     ↓
              의약외품 품목허가 + 출시
```

## 의약외품 분류 바운더리

| 항목 | 범위 |
|---|---|
| 법적 근거 | 약사법 + 식약처 의약외품 품목허가·신고·심사 규정 |
| 허용 클레임 | 예방·완화·청결·각성 (구체 효능은 카테고리별) |
| 시험성적서 | 살균력·흡수량·안전성·자극성·VOC (카테고리별) |
| 광고 금기 | "치료" 표현 (의약품 영역) · "최고/끝판" 절대적 표현 (광고정지 사례 잇치) |
| GMP | 의약외품 GMP (2023.9 시행) — 화장품 GMP와 별도 시설 |

### ⚠ 재분류 이력

- 2018.7 식약처 가이드라인 — **탈모방지 샴푸 → 기능성 화장품** 재분류 (의약외품 신규 진입 불가)
- 대안: **항비듬·항진균·각질·피지조절** 클레임은 의약외품 잔존
- 따라서 QD-HSHAMPOO/HSPRAY는 "항비듬·두피환경 개선" 클레임으로 R&D (사용자 인지는 "탈모케어 풀라인"으로 연결)

## 공통 측정 방법

| 영역 | 측정 |
|---|---|
| 항균 | MIC (Minimum Inhibitory Concentration) · log reduction · BS EN 1276 |
| 살균 | Time-kill kinetics · ASTM E2315 |
| 항진균 | MIC (말라세지아·칸디다) · CLSI M27/M44 |
| 흡수 | g/매 · Franz cell (모낭 침투 깊이) · 통기성 g/m²·24hr |

| 효능 | 카테고리별 임상 (모근·치주염·각성·기피력·VSC 등) |

| 잔류 | GC-MS · HPLC |
| 입자 차단 | NaCl 0.3μm aerosol · KF 등급 시험 |
