\section{Reproducibility Runbook}
\label{app:repro}

This appendix is the bit-for-bit reproduction runbook. Every Stage-1--3
atom is recomputable from the top-level CLI ---
\verb|hexa verify --expr ioffe_loop_bz <a> <I> <zeta> <expected>|,
\verb|... triple_product <J> <B> <V> <F>|,
\verb|... pair_threshold_total <mpc2> <expected>|, and
\verb|... rel_kinetic_from_p <pc> <expected>| --- all at
$\varepsilon=10^{-9}$, $|\Delta|=0$, and all already folded into the
atlas SSOT (\verb|compiler/atlas/embedded.gen.hexa|, hash
\texttt{663698a0}) by their parent domains, so UFO's reuse is an
idempotent skip. The Stage-4--7 lattice identities are checked via
\verb|UFO/verify/lattice_check.hexa| and \verb|numerics_*.hexa|; the
4-layer digital-twin closed-form anchors ship as \verb|.hexa| decks
under \verb|UFO/sim/decks/| (thermal-cryo and F-ANTI-3 closure, both
9/9 PASS). The deferred heavy body-solves (CFD, EM 3D, FEA,
$\circlearrowleft$) are documented as pool/cloud micro-experiments
(@D~d7). Paper build: \texttt{make} (xelatex $\times3$ $+$ bibtex);
\texttt{make figures}; \texttt{make pages}; \texttt{make lint}. The
\texttt{companion/} axis externalizes every record:
\texttt{verify-ledger.json}, \texttt{pr-roll.json},
\texttt{integration-map.json}, and \texttt{session-journal.md}.

\subsection{Per-atom recompute commands}
\label{app:repro-atoms}

The nine Stage-1--3 \tierGreen\ atoms reproduce exactly with these CLI
calls (host \texttt{mini}, macOS libm; ubu-2 segfaults and ubu-1 is
broken, so pin \texttt{mini}):

\begin{lstlisting}
# Stage-1 hover (RTSC ioffe_loop_bz, B = mu0*I/(2a) = 2 T)
hexa verify --expr ioffe_loop_bz 0.3 954930.0 0.0 2.0     # |Delta|=5.67e-10
hexa verify --expr ioffe_loop_bz 0.5 1591549.0 0.0 2.0    # |Delta|=5.67e-10
hexa verify --expr ioffe_loop_bz 1.0 3183099.0 0.0 2.0    # |Delta|=5.67e-10
# Stage-2 cruise (FUSION triple_product, F = J*B*V)
hexa verify --expr triple_product 200000.0 5.0 0.096 96000.0   # |Delta|=0
hexa verify --expr triple_product 150000.0 3.0 0.096 43200.0   # |Delta|=0
hexa verify --expr triple_product 100000.0 2.0 0.096 19200.0   # |Delta|=0
# Stage-3 orbital (ANTIMATTER annihilation + relativistic exhaust)
hexa verify --expr pair_threshold_total 938.272 6567.904       # |Delta|=0
hexa verify --expr rel_kinetic_from_p 9382.72 8491.244898780338 # |Delta|=0
hexa verify --expr rel_kinetic_from_p 1625.13 938.272          # |Delta|=0
\end{lstlisting}

Each returns \texttt{tier = SUPPORTED-NUMERICAL} and
\texttt{absorb = already in atlas -- idempotent skip} (the atoms were
folded by RTSC/FUSION/ANTIMATTER, @D~g69). The lattice \tierBlue\
identities recompute via:

\begin{lstlisting}
hexa verify --expr lattice_sigma_tau 6 48   # sigma(6)*tau(6)=12*4=48
hexa verify --expr lattice_sigma_phi 6 24   # sigma(6)*phi(6)=12*2=24=n*tau
hexa run UFO/verify/lattice_check.hexa       # all n=6 integer identities
\end{lstlisting}

\subsection{Atlas SSOT provenance}
\label{app:repro-atlas}

The live atlas hash is verified with \verb|hexa atlas hash| (expected
\texttt{663698a0\dots ce7fb}, 16134 nodes). The Stage-1--3 atoms resolve
under their \texttt{verified-<fn>-num} keys:

\begin{lstlisting}
hexa atlas hash                                       # -> 663698a0...ce7fb
hexa atlas lookup verified-pair_threshold_total-num   # embedded.gen.hexa:30652
hexa atlas lookup verified-rel_kinetic_from_p-num     # embedded.gen.hexa:30514
hexa atlas lookup --prefix=verified-triple            # triple_product:30535
\end{lstlisting}

UFO registers \emph{no new atoms} --- every Stage-1--3 atom was folded by
its parent domain, so the hash is stable across the capstone's PRs
(idempotent reuse, @D~d3).

\subsection{Paper build and the companion axis}
\label{app:repro-build}

The monograph builds with three xelatex passes and a bibtex pass:

\begin{lstlisting}
# from ufo-capstone/ (xelatex: UTF-8/emoji-native; tier badges are TikZ discs)
xelatex -interaction=nonstopmode main.tex
bibtex main
xelatex -interaction=nonstopmode main.tex
xelatex -interaction=nonstopmode main.tex
# figures: each figures/_scripts/figNN_*.tex -> figures/figNN_*.pdf (pdflatex)
# convenience targets:
make figures   # regenerate all standalone pgfplots figures
make pages     # page count via pdfinfo
make lint      # /paper lint . (commons @D g51)
\end{lstlisting}

The \texttt{companion/} axis externalizes every record so a reader can
audit without re-running: \texttt{verify-ledger.json} (the Stage-1--3
\tierGreen\ atoms + lattice \tierBlue\ identities + the open \tierWhite\
falsifiers), \texttt{pr-roll.json} (the landed UFO PRs with role
metadata), \texttt{integration-map.json} (the stage$\to$sibling-domain
map of \S\ref{sec:intmap}), and \texttt{session-journal.md} (the capstone
narrative).

\subsection{Honest scope}
\label{app:repro-scope}

Everything in this runbook reproduces exactly \emph{except} the two
\tierOrange\ deferred body-solves (CFD, $\circlearrowleft$), which require
pool/cloud heavy compute (@D~d7) and are not yet converged --- the reason
\texttt{absorbed=false}. The Stage-4--7 lattice identities reproduce as
integer checks, but the physics they index is UNPROVEN
(App.~\ref{app:stage47}); recomputing the integers does not reproduce a
warp bubble or a traversable throat. The runbook reproduces the
\emph{verified} content and \emph{documents} the deferred and unproven
content --- exactly the honesty contract the capstone is built on.
